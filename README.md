## 🚀 Fitur Utama

Aplikasi ini memiliki lebih dari 16 fitur utama yang disusun dalam sistem tab yang rapi dan responsif:

### 📁 Manajemen File & Sistem
* **File Explorer**: Navigasi direktori secara cepat, edit file teks, ganti nama, hingga menghapus file/folder secara permanen.
* **Task Manager**: Memantau penggunaan sumber daya perangkat keras (CPU & RAM) dari proses aktif secara real-time.
* **Global Search**: Mencari file di seluruh sistem penyimpanan (seperti Drive C:) tanpa menghambat kinerja aplikasi (Anti-Freeze).
* **File Splitter**: Memecah file berukuran besar menjadi beberapa bagian kecil (1MB per part) untuk memudahkan pengiriman data.

### 🔐 Keamanan & Enkripsi
* **Password Generator**: Menciptakan kata sandi kuat 16 karakter dengan algoritma acak yang aman.
* **File Encryptor**: Mengamankan dokumen sensitif menggunakan teknik XOR Encryption terintegrasi.
* **Hash Generator**: Verifikasi integritas file atau teks dengan nilai MD5, SHA1, dan SHA256.
* **Base64 Tool**: Konversi teks ke format Base64 untuk kebutuhan enkoding data yang efisien.

### 🌐 Web & Jaringan
* **Web Source Downloader**: Mengambil dan menganalisis kode sumber (HTML) dari alamat website manapun secara instan.
* **Terminal Emulator**: Menjalankan perintah sistem (CMD/PowerShell) secara langsung dari dalam antarmuka aplikasi.
* **Speedtest Simulator**: Melakukan pengujian kecepatan unduh dan unggah jaringan secara virtual.

### ⚙️ Utilitas Tambahan
* **World Clock**: Pantau waktu global secara akurat di berbagai zona waktu utama seperti Jakarta, Beijing, Moscow, dan London.
* **Unit Converter Pro**: Konversi berbagai satuan berat, suhu (Celsius/Fahrenheit), hingga kalkulasi kapasitas data (MB/GB).
* **ASCII Art Gen**: Mengubah teks biasa menjadi seni visual karakter yang unik.
* **Color Picker**: Memilih dan mengambil kode warna HEX secara presisi untuk kebutuhan desain.
